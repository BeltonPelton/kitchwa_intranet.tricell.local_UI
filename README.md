# User Interface Design

## Overview

This repository contains an example of a web application using JavaScript and Node.js.

It is a student project within the university course **"User Interface Design" (5 ECTS)** offered by Åland University of Applied Sciences.

The purpose was to learn how to design and implement a modern, responsive user interface for a web application. This project involves the construction of five different versions.
Those being: css-grid, html5, multimedia, responsive and bootstrap. This lets us experiment and learn about multiple ways of achieving the same goal.

This project builds upon the web application developed in the two previous courses, **"Frontend Development" (5 ECTS)** and **"Backend
Development" (5 ECTS)**, making this course a natural continuation of earlier work.

---

## Project Structure

/config – Configuration files

/data – Data sources in various formats for exercises (JSON, XML, MDB)

/masterframe – Frontend HTML files

/public/css – Stylesheet files

/public/images – Layout images (GIF, JPG, PNG)

/public/photos – Personnel registry images

/public/scripts – Frontend JavaScript files

/public/t-veronica, tyrant, uroboros - Files for object observation

/public/virusphoto - Storage for submitted pictures for each object

/public/text - Text information to be displayed on the application

/routes – Route definitions (JavaScript)

/ – Main application files
